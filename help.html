The Ultimate Guide to the Quizdown Suite

Welcome to your all-in-one solution for creating powerful educational materials. This guide covers every feature at your disposal, from writing your first question to generating professional, print-ready exams. Unleash your teaching potential.

The Header: Your Quiz's Control Panel

Every great document starts with a great title. Your quiz is no different. Begin every file with a header section enclosed in `---` to control your quiz's metadata. The title is the sole required field in the header section.

---
title: Final Exam
---

Building a Professional Cover Page (LaTeX/PDF)

The header is your command center for creating a beautiful, custom cover page for printable exams. The system is designed to be completely intuitive: what you type is what you get.

The Mandatory Title

The `Title` key is special. You can type `Title:`, `title:`, or even `TITLE:`, and the system will intelligently use it as the main title for everything: the cover page, the browser tab, and the filename.

Unlimited Custom Fields

For all other information, you have complete freedom. Simply type the label you want, followed by a colon. The converter will use your exact wording and capitalization on the cover page.

There are no predefined fields—create exactly what you need.

The Anatomy of a Question

Separate each question with `---`. Within each question block, you have a flexible set of tools to create the perfect assessment.

Question Types

  • Multiple-Choice: The classic question type. Mark the correct answer with `[x]` and incorrect ones with `[ ]`. The options are automatically shuffled in the final quiz to ensure fairness.
  • Open-Ended: For questions requiring detailed, written answers. The question text starts after `#Q` and the complete explanation starts after `#A`. The answer is hidden behind a "Show/Hide" button in the interactive quiz.

Assigning Points (LaTeX/PDF)

For printable exams, you can assign point values to questions. This adds a score marker (e.g., "___ / 5 p.") to the right of the question number in the generated LaTeX/PDF file. This feature is for print output only and will not appear in the interactive HTML quiz.

To assign points, simply add a colon and the number of points after the #Q marker on the same line.

#Q:5
What is the capital of France?
- [x] Paris
- [ ] London
- [ ] Berlin
#A
Paris is the capital of France.

If you use a standard #Q with no points, no marker will be displayed.

Text Formatting

Clarity is key. Use these simple markers to format your text.

  • Bold Text: Wrap text in double asterisks: `**your bold text**`.
  • Italic Text: Wrap text in single asterisks: `*your italic text*`.

Mathematical and Scientific Notation (LaTeX)

Write beautiful, crystal-clear mathematical expressions. This feature works perfectly for both on-screen HTML and printed PDF documents.

  • Inline Math: For expressions within a sentence, like $E=mc^2$, use single dollar signs: `$E=mc^2$`.
  • Block Math: For larger, centered equations, use double dollar signs.
$$
\frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)
$$

Adding Rich Content: The Material Blocks

Go beyond simple text. Embed rich, formatted content directly into your questions to provide context, data, or source material. All material blocks are placed inside the question body (after the `#Q` line).

General Information: `[material]`

Use this for passages of text, instructions, or any general context the student needs to answer the question.

[material]
Read this historical account carefully. It contains clues that will help you answer the questions below.
[/material]

Code Snippets: `[code]`

Perfect for computer science, programming, or logic questions. Code is displayed in a clean, monospaced font, preserving all indentation and spacing.

[code]
function factorial(n) {
  if (n === 0) {
    return 1;
  }
  return n * factorial(n - 1);
}
[/code]

Quotes: `[quote]`

Include quotes from literature, historical figures, or academic sources. You can provide an attribution with a clear label such as Author:, Source:, or By:.

[quote]
The measure of intelligence is the ability to change.
Author: Albert Einstein
[/quote]

Data Tables: `[table]`

Present structured data cleanly and effectively. Use pipes `|` to separate columns and a `|---|` line to separate the header from the table body. Essential for questions in science, finance, and social studies.

[table]
| Asset Class | Risk Profile | Liquidity |
|---|---|---|
| Government Bonds | Low | High |
| Real Estate | Medium-High | Low |
[/table]

Your Output Suite: From Screen to Print

One quiz file, multiple professional outputs. This is where the magic happens.

1. Interactive HTML Quizzes

The "Generate HTML" button creates a fully interactive, self-contained quiz that runs in any web browser, even offline. It's responsive, accessible, and perfect for online learning. Use the "Language" dropdown to translate the interface buttons (`Check`, `Show/Hide`) into English or Finnish for your audience.

2. Professional LaTeX & PDF Documents

The second row of buttons is your gateway to creating print-ready exam papers and answer keys.

  • Generate Questions: Creates a LaTeX file (`.tex`) with only the questions, perfectly formatted for a live exam. This includes your professional cover page if you've added the metadata.
  • Generate Questions & Answers: Creates a complete document with the questions, followed by a separate, clearly marked answer key section. Ideal for grading or creating study guides.

You can copy the LaTeX code directly or download the `.tex` file to compile into a PDF using any standard LaTeX editor.

Your Personal Quiz Library

Never lose your work again. The converter includes a powerful library to save, load, and manage all your quizzes directly in your browser.

  • Save Quiz: Click to instantly save the entire state of your work—the Quizdown content, the CSS styles, and the JS script. A unique title is generated automatically if a quiz with the same name exists.
  • Load & Manage: Your saved quizzes appear in a list sorted by date. Click "Load" to instantly bring a quiz back into the editors.
  • Powerful Search: Use the search bar to find quizzes by title or by any word or phrase *inside the quiz content itself*.
  • Safe Deletion: Click "Delete" once to prime it, then a second time ("Sure?") to confirm. This prevents accidental deletion of your hard work.

#Q 
Prove that $\sqrt{2}$ is irrational.
#A
Assume $\sqrt{2}$ is rational, i.e., $\sqrt{2} = \frac{a}{b}$ where $a, b \in \mathbb{Z}$ with no common factors and $b \ne 0$.


Then,
$$2 = \frac{a^2}{b^2} \Rightarrow a^2 = 2b^2$$


Since $a^2$ is even, $a$ must be even. Let $a = 2k$ for some integer $k$.


Substitute back:
$$(2k)^2 = 2b^2 \Rightarrow 4k^2 = 2b^2 \Rightarrow b^2 = 2k^2$$


So $b^2$ is even, implying $b$ is even.


But then both $a$ and $b$ are even, contradicting the assumption.


Therefore, $\sqrt{2}$ is irrational.
