The Anatomy of a Question

Separate each question with `---`. Within each question block, you have a flexible set of tools to create the perfect assessment.

Question Types

  • Multiple-Choice: The classic question type. Mark the correct answer with `[x]` and incorrect ones with `[ ]`. The options are automatically shuffled in the final quiz to ensure fairness.
  • Open-Ended: For questions requiring detailed, written answers. The question text starts after `#Q` and the complete explanation starts after `#A`. The answer is hidden behind a "Show/Hide" button in the interactive quiz.

Assigning Points (LaTeX/PDF)

For printable exams, you can assign point values to questions. This adds a score marker (e.g., "___ / 5 p.") to the right of the question number in the generated LaTeX/PDF file. This feature is for print output only and will not appear in the interactive HTML quiz.

To assign points, simply add a colon and the number of points after the #Q marker on the same line.

#Q:5
What is the capital of France?
- [x] Paris
- [ ] London
- [ ] Berlin
#A
Paris is the capital of France.

If you use a standard #Q with no points, no marker will be displayed.

Text Formatting

Clarity is key. Use these simple markers to format your text.

  • Bold Text: Wrap text in double asterisks: `**your bold text**`.
  • Italic Text: Wrap text in single asterisks: `*your italic text*`.

Mathematical and Scientific Notation (LaTeX)

Write beautiful, crystal-clear mathematical expressions. This feature works perfectly for both on-screen HTML and printed PDF documents.

  • Inline Math: For expressions within a sentence, like $E=mc^2$, use single dollar signs: `$E=mc^2$`.
  • Block Math: For larger, centered equations, use double dollar signs.
$$
\frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)
$$

Adding Rich Content: The Material Blocks

Go beyond simple text. Embed rich, formatted content directly into your questions to provide context, data, or source material. All material blocks are placed inside the question body (after the `#Q` line).

General Information: `[material]`

Use this for passages of text, instructions, or any general context the student needs to answer the question.

[material]
Read this historical account carefully. It contains clues that will help you answer the questions below.
[/material]

Code Snippets: `[code]`

Perfect for computer science, programming, or logic questions. Code is displayed in a clean, monospaced font, preserving all indentation and spacing.

[code]
function factorial(n) {
  if (n === 0) {
    return 1;
  }
  return n * factorial(n - 1);
}
[/code]

Quotes: `[quote]`

Include quotes from literature, historical figures, or academic sources. You can provide an attribution by using an em dash (`—`).

[quote]
The measure of intelligence is the ability to change.
— Albert Einstein
[/quote]

Data Tables: `[table]`

Present structured data cleanly and effectively. Use pipes `|` to separate columns and a `|---|` line to separate the header from the table body. Essential for questions in science, finance, and social studies.

#Q 
Prove that $\sqrt{2}$ is irrational.
#A
Assume $\sqrt{2}$ is rational, i.e., $\sqrt{2} = \frac{a}{b}$ where $a, b \in \mathbb{Z}$ with no common factors and $b \ne 0$.


Then,
$$2 = \frac{a^2}{b^2} \Rightarrow a^2 = 2b^2$$


Since $a^2$ is even, $a$ must be even. Let $a = 2k$ for some integer $k$.


Substitute back:
$$(2k)^2 = 2b^2 \Rightarrow 4k^2 = 2b^2 \Rightarrow b^2 = 2k^2$$


So $b^2$ is even, implying $b$ is even.


But then both $a$ and $b$ are even, contradicting the assumption.


Therefore, $\sqrt{2}$ is irrational.

#Q 
Formulate the Riemann Hypothesis using the Riemann zeta function $\zeta(s)$, explain its connection to prime number distribution, and calculate the exact value of $\zeta(2)$ using Euler's product formula. Include all critical components: analytic continuation, non-trivial zeros, and the significance of the critical line $\text{Re}(s) = \frac{1}{2}$.




[quote]
"Mathematics is the queen of the sciences, and number theory is the queen of mathematics."
— Carl Friedrich Gauss
[/quote]




[material]
The Riemann zeta function $\zeta(s)$ is initially defined for $\text{Re}(s) > 1$ as an infinite series. Through complex analysis techniques, it can be extended to other regions of the complex plane. This function has certain points where its value equals zero, which are categorized into two types. The distribution of prime numbers shows surprising connections to the behavior of this function. Leonhard Euler discovered a remarkable relationship between this function and prime numbers in the 18th century.
[/material]




[code]
# Python snippet for zeta function approximation
import math
def zeta_approx(s, terms=1000):
    return sum(1/(n**s) for n in range(1, terms+1))




# Example: zeta_approx(2, 1000) ≈ 1.64393...
[/code]




#A
The Riemann zeta function $\zeta(s)$ is initially defined for $\operatorname{Re}(s)>1$ by $\displaystyle \zeta(s)=\sum_{n=1}^\infty n^{-s}$. It admits a meromorphic continuation to the whole complex plane with a simple pole at $s=1$. The trivial zeros occur at the negative even integers $s=-2,-4,\dots$; all other zeros are called non-trivial zeros and lie in the critical strip $0<\operatorname{Re}(s)<1$. The Riemann Hypothesis asserts that every non-trivial zero satisfies $\operatorname{Re}(s)=\tfrac12$.


This hypothesis is deeply connected to prime distribution through Euler's product formula: $\zeta(s) = \prod_{p \text{ prime}} \frac{1}{1 - p^{-s}}$ for $\operatorname{Re}(s) > 1$. Taking logarithms yields $\ln \zeta(s) = \sum_p \sum_{k=1}^{\infty} \frac{1}{k p^{ks}}$, linking $\zeta(s)$ to prime counting. The explicit formula for $\psi(x) = \sum_{n \leq x} \Lambda(n)$ (where $\Lambda$ is the von Mangoldt function) is:


$$\psi(x) = x - \sum_{\rho} \frac{x^{\rho}}{\rho} - \ln(2\pi) - \frac{1}{2} \ln(1 - x^{-2})$$


where $\rho$ ranges over non-trivial zeros. If RH holds, the error term $\left| \psi(x) - x \right|$ is bounded by $O(\sqrt{x} \ln^2 x)$.


For $\zeta(2)$, Euler's product gives:
$$\zeta(2) = \prod_{p} \frac{1}{1 - p^{-2}} = \left( \frac{1}{1 - 2^{-2}} \right) \left( \frac{1}{1 - 3^{-2}} \right) \left( \frac{1}{1 - 5^{-2}} \right) \cdots$$


Evaluating the infinite product or using Fourier series, $\zeta(2) = \frac{\pi^2}{6} \approx 1.644934$.


Current evidence includes verification of over $10^{13}$ non-trivial zeros on the critical line, but no general proof. RH remains one of the Clay Mathematics Institute's Millennium Prize Problems.

#Q 
Calculate the exact value of $\zeta(0)$ using the functional equation of the Riemann zeta function. Given the functional equation:
$$\zeta(s) = 2^s \pi^{s-1} \sin\left(\frac{\pi s}{2}\right) \Gamma(1-s) \zeta(1-s)$$
- [x] $-\frac{1}{2}$
- [ ] $0$
- [ ] $1$
- [ ] $\frac{1}{2}$
- [ ] Undefined
#A 
To calculate $\zeta(0)$ using the given functional equation:


$$\zeta(s) = 2^s \pi^{s-1} \sin\left(\frac{\pi s}{2}\right) \Gamma(1-s) \zeta(1-s)$$


we consider the limit as $s \to 0$, since direct substitution leads to an indeterminate form due to the pole of $\zeta(1-s)$ at $s=0$.


As $s \to 0$:
- $2^s \to 1$,
- $\pi^{s-1} \to \pi^{-1}$,
- $\sin\left(\frac{\pi s}{2}\right) \sim \frac{\pi s}{2}$,
- $\Gamma(1-s) \to \Gamma(1) = 1$,
- $\zeta(1-s) \sim \frac{1}{-s}$ (since $\zeta$ has a simple pole at $1$ with residue $1$, and $1-s \to 1$ from below).


Multiplying these approximations:
$$\zeta(s) \sim 1 \cdot \frac{1}{\pi} \cdot \frac{\pi s}{2} \cdot 1 \cdot \left(-\frac{1}{s}\right) = -\frac{1}{2}$$


Taking the limit:
$$\zeta(0) = \lim_{s \to 0} \zeta(s) = -\frac{1}{2}$$


This result follows rigorously from the functional equation and the analytic structure of $\zeta(s)$. No external summation methods or unrelated zero properties are needed.


Final answer: $-\frac{1}{2}$

#Q 
Analyze the relationship between the Riemann Hypothesis and the Mertens function $M(x) = \sum_{n \leq x} \mu(n)$, where $\mu(n)$ is the Möbius function. Discuss the implications of RH on $M(x)$ and evaluate the truth of the statement: "RH implies $M(x) = O(x^{1/2 + \epsilon})$ for all $\epsilon > 0$."




[table]
| Concept | Definition | RH Implication |
|---------|------------|----------------|
| Möbius function $\mu(n)$ | $\mu(n) = \begin{cases} 1 & n=1 \\ (-1)^k & n \text{ square-free with } k \text{ prime factors} \\ 0 & \text{otherwise} \end{cases}$ | Critical for series $\sum \mu(n)/n^s = 1/\zeta(s)$ |
| Mertens function $M(x)$ | $M(x) = \sum_{n \leq x} \mu(n)$ | Bound directly tied to zero-free region of $\zeta(s)$ |
| Equivalent RH statement | $M(x) = O(x^{1/2 + \epsilon}) \forall \epsilon > 0$ | Sharp bound confirmed if RH true |
| Known unconditional bound | $M(x) = O\left(x \exp\left(-c \frac{(\ln x)^{3/5}}{(\ln \ln x)^{1/5}}\right)\right)$ | Weaker than RH prediction |
[/table]




#A
The Mertens function $M(x) = \sum_{n \leq x} \mu(n)$ is intrinsically linked to the Riemann zeta function via the identity $\frac{1}{\zeta(s)} = \sum_{n=1}^{\infty} \frac{\mu(n)}{n^s}$ for $\operatorname{Re}(s) > 1$. The Riemann Hypothesis (RH) implies that all non-trivial zeros of $\zeta(s)$ lie on $\operatorname{Re}(s) = \frac{1}{2}$, which directly affects the growth rate of $M(x)$.


Specifically, RH is equivalent to the statement that $M(x) = O(x^{1/2 + \epsilon})$ for every $\epsilon > 0$. This means that for any $\epsilon > 0$, there exists a constant $C_\epsilon > 0$ such that $|M(x)| \leq C_\epsilon x^{1/2 + \epsilon}$ for all sufficiently large $x$. The exponent $\frac{1}{2} + \epsilon$ is optimal—if RH fails, the best possible bound would be worse (e.g., $O(x^{\sigma})$ where $\sigma > \frac{1}{2}$ is the supremum of real parts of zeros).


The equivalence arises from Perron's formula and contour integration: 
$$M(x) = \frac{1}{2\pi i} \int_{c - i\infty}^{c + i\infty} \frac{x^s}{s \zeta(s)}  ds \quad (c > 1)$$


Shifting the contour left past the critical line, the error term depends on the zero-free region. If all zeros satisfy $\operatorname{Re}(s) \leq \frac{1}{2}$, the integral yields $M(x) = O(x^{1/2 + \epsilon})$.


Unconditionally (without RH), the best bound is $M(x) = O\left(x \exp\left(-c (\ln x)^{3/5} (\ln \ln x)^{-1/5}\right)\right)$ (Walfisz, 1963), which grows faster than any $x^{1/2 + \epsilon}$. Numerical evidence shows $|M(x)| < \sqrt{x}$ for $x < 10^{16}$, but this fails infinitely often (Odlyzko & te Riele, 1985). Thus, the statement "RH implies $M(x) = O(x^{1/2 + \epsilon})$ for all $\epsilon > 0$" is **true** and is a well-established equivalence in analytic number theory.

These claims were forcefully presented to the international community, most notably in Secretary of State Colin Powell's February 2003 address to the UN Security Council, where he asserted the intelligence was based on "facts and conclusions." The administration argued that Iraq was actively producing chemical and biological weapons and seeking nuclear capabilities, thereby violating UN Security Council resolutions.


However, these justifications stood in stark contrast to the findings of UN weapons inspectors. The final pre-invasion inspections conducted by UNMOVIC and the IAEA between November 2002 and March 2003 found no conclusive evidence of active WMD programs. Inspectors reported that while they visited hundreds of sites, Iraq's cooperation was uneven, with access delayed or restricted at certain facilities and key disarmament questions remaining unresolved. This contradicted the certainty expressed by U.S. officials. After the invasion, the Iraq Survey Group, a U.S.-led fact-finding mission, concluded that while Saddam Hussein intended to reconstitute WMD programs, no stockpiles existed at the time of the invasion, which ultimately revealed a significant pre-war intelligence failure. The discrepancy between the administration's claims and the on-the-ground reality fueled a major international controversy over the legality and necessity of the war.

#Q
For the simple linear regression model $y_i = \beta_0 + \beta_1 x_i + \epsilon_i$, derive the least-squares estimate of the slope $\hat{\beta}_1$ by minimizing the sum of squared residuals.
#A
The objective is to minimize the sum of squared residuals (SSR), $S(\beta_0, \beta_1) = \sum_{i=1}^{n} (y_i - \beta_0 - \beta_1 x_i)^2$. To find the minimum, we take the partial derivatives with respect to $\beta_0$ and $\beta_1$ and set them to zero.


$$
\frac{\partial S}{\partial \beta_0} = -2 \sum (y_i - \beta_0 - \beta_1 x_i) = 0 \implies \sum y_i - n\beta_0 - \beta_1 \sum x_i = 0
$$


$$
\frac{\partial S}{\partial \beta_1} = -2 \sum x_i(y_i - \beta_0 - \beta_1 x_i) = 0 \implies \sum x_i y_i - \beta_0 \sum x_i - \beta_1 \sum x_i^2 = 0
$$


From the first equation, we get the estimator for the intercept: $\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}$. Substituting this into the second equation and solving for $\hat{\beta}_1$ yields:


$$
\hat{\beta}_1 = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2} = \frac{\text{Cov}(x, y)}{\text{Var}(x)}
$$

#Q
Derive the maximum efficiency $\eta_{\text{Carnot}}$ of a reversible Carnot engine operating between a hot reservoir at temperature $T_H$ and a cold reservoir at temperature $T_C$.
#A
The efficiency $\eta$ of any heat engine is defined as the ratio of the work done, $W$, to the heat absorbed from the hot reservoir, $Q_H$.


$$ \eta = \frac{W}{Q_H} $$


From the first law of thermodynamics, $W = Q_H - Q_C$, where $Q_C$ is the heat rejected to the cold reservoir.


$$ \eta = \frac{Q_H - Q_C}{Q_H} = 1 - \frac{Q_C}{Q_H} $$


For a reversible cycle, such as the Carnot cycle, the ratio of the heats is equal to the ratio of the absolute temperatures of the reservoirs (a consequence of the second law of thermodynamics):


$$ \frac{Q_C}{Q_H} = \frac{T_C}{T_H} $$


Substituting this into the efficiency equation gives the Carnot efficiency:


$$ \eta_{\text{Carnot}} = 1 - \frac{T_C}{T_H} $$


This represents the maximum possible efficiency for any heat engine operating between these two temperatures.
